# Encryptie
Deze kleine applicatie kan een text encrypteren en decrypteren met behulp van verschillende algoritmes. Met behulp van de rabiobuttons kies je het algoritme.
Er is enkel een achterstevoren algoritme geïmplementeerd voor demonstratiedoeleinden die de tekst omkeert van richting. De andere algoritmes zijn nog niet geïmplementeerd en switchen tussen de algoritmes werkt nog niet.

## Opgave
* Implementeer een ander algoritme naar keuze. Bijvoorbeeld ROT13, vervangen van elke letter door een cijfer, ...
* Gebruik interfaces om de code abstract te houden en gemakkelijk uit te breiden met nieuwe algoritmes.
* Enkel in RadioButton_Checked mag je code toevoegen om te switchen tussen de algoritmes. Enkel daar mag je code gebruiken die kennis heeft van de verschillende algoritmes.
